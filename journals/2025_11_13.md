## Win7使用ParaView
collapsed:: true

- 4.3版本ParaView似乎无法正确读取xdmf文件，win7最高支持ParaView5.9.1，这个版本可以正常读取xdmf文件。[XDMF, Windows 7, paraview 4.3.1 carshing a lot - ParaView Support - ParaView](https://discourse.paraview.org/t/xdmf-windows-7-paraview-4-3-1-carshing-a-lot/16276#_prevue), [Win 7 Installation problem - ParaView Support - ParaView](https://discourse.paraview.org/t/win-7-installation-problem/9313)
- 虚拟机中无法正常使用ParaView，需要手动添加Mesa库。下载[Mesa库部署包](https://github.com/pal1000/mesa-dist-win/releases)中的“release-mingw”包，运行“perappdeploy.cmd”，回车确认，拖入ParaView安装目录下的bin文件夹回车确认，按y确认，输入paraview回车确认，输入2选择x64选项，回车确认，后续按n直到窗口关闭，此后打开ParaView发现可用。 [How to run ParaView (including the binary release) using Mesa on Windows - Tips and Tricks - ParaView](https://discourse.paraview.org/t/how-to-run-paraview-including-the-binary-release-using-mesa-on-windows/4321)

- 续接V3工作流（详见“扫描版PDF转EPUB_V3”条目），新建文件夹并新建md，从Sigil中复制章节内容到新建的md文件
- 在VSCode中打开该文件，在`<p>`前增加空行隔开段落（否则会被校对工具识别为一段），使用插件“AI Proofreader”拆分md文件并校对（详见“AI辅助文本校对”条目），打开差异文件，逐项检查，可修改绿色部分。
	- 校对新章节时可以直接复制到之前打开的md文件中，无需重新导入，直接“重新拆分”即可
- 修改完毕，复制proofread.json.md文件内容粘贴回Sigil对应章节，保存
- 对特殊页面套用对应样式，对BOOKpandoc自带样式做一些修改
  ``` css
  body> :not(.title-page):not(.dedication):not(#TOC):not(.toc):not(#title-block-header) {
      padding-left: 2cm;
      padding-right: 2cm;
      padding-bottom: 2cm;
  }
  .divider {
      page-break-before: always;
      break-before: page;
      text-align: center;
      margin: 3em 0;
      color: var(--mid-gray);
      font-size: 0.75rem;
      letter-spacing: 1em;
      page-break-inside: avoid;
  }
  a {
      color: #0066cc;
      text-decoration: none;
  }
  ```
- 使用 [Readest — 免费开源 EPUB 和 PDF 电子书阅读器，全平台支持](https://readest.com/zh) 查看EPUB，注意修改内容后需要重新导入才能生效（不需要删除）
- 对版权页我使用blockquote样式，目录使用TOC样式，前言使用dedication样式，后记标题使用epilogue，注释区域前使用divider样式分割

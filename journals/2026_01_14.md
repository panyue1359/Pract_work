## C盘空间清理之WSL子系统
collapsed:: true
	- ((65030192-d6e7-4fac-823d-a9ede88df32a))
- 将当前安装在C盘的子系统导出备份，然后安装备份到其他盘，最后删除C盘的子系统并重新注册用户 [WSL2 Ubuntu-20.04文件太占c盘空间，使用导入导出来实现位置挪移，移动到d盘下面 - 知乎](https://zhuanlan.zhihu.com/p/510650353), [WSL2 安装、卸载以及常用命令一览 – Dancying's Study Notes](https://dancying.cn/windows/windows-003.html)
  ``` cmd
  wsl -l -v
  wsl --shutdown 子系统名称
  mkdir 备份文件夹
  wsl --export 子系统名称 备份文件夹\备份文件名.bak
  wsl --import 新的子系统名称 新的子系统安装文件夹 备份文件路径 --version 2
  wsl --unregister 旧的子系统名称
  ubuntu config --default-user 用户名
  ```

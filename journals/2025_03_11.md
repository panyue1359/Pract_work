## VS Code添加OpenFOAM语法高亮
collapsed:: true
	- ((68274234-f87f-4ce0-865f-2164130b5fd0))
- Aref的“OpenFOAM Dictionary Support”插件：支持OpenFOAM语法高亮、鼠标悬停提示、自动补全、看板展示配置文件内容，粘贴以下文件关联选项
  ``` json
  "files.associations": {
          "*Dict": "openfoam",
          "*Properties": "openfoam",
          "fvSchemes": "openfoam",
          "fvSolution": "openfoam",
          "**/constant/g": "openfoam",
          "**/0/*": "openfoam",
          "**/system/*": "openfoam",
          "**/constant/*/polyMesh/*": "openfoam",
          "**/0/*/*": "openfoam",
          "**/0.orig/*": "openfoam",
          "**/0.orig/*/*": "openfoam"
      },
  ```
- Zhikui Guo的“OpenFOAM”插件：按照提示打开设置，查找“json”，点击“Edit in settings.json”打开配置文件，在其中粘贴文件关联选项。高亮关键词可以自动补全，但是字典不太完整，且已经停止更新
  ``` json
  "files.associations": {
          "*Dict": "OpenFOAM",
          "*Properties": "OpenFOAM",
          "fvSchemes": "OpenFOAM",
          "fvSolution": "OpenFOAM",
          "**/constant/g": "OpenFOAM",
          "**/0/*": "OpenFOAM",
          "**/system/*": "OpenFOAM",
          "**/constant/*/polyMesh/*": "OpenFOAM",
          "**/0/*/*": "OpenFOAM",
          "**/0.orig/*": "OpenFOAM",
          "**/0.orig/*/*": "OpenFOAM"
      },
  ```

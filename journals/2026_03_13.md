- 实际上是给CLI做了个界面，组件由“智能体+技能（MCP/Skills）+工作空间”组成，相比聊天机器人，还普遍具有定时任务、控制浏览器、文件预览功能，但是不能让大模型重新回答问题或者生成聊天分支，不满意只能重开。所以，这类客户端的定位不是聊天（如 Cherry Studio、Chatbox等），而是使用大模型执行预先定义好的工作流。
- 我很关心技能管理问题，大部分这类型软件会建立一个总站储存所有技能，然后在特定的工作空间或者智能体中开启或者关闭部分技能，类似于Claude Code的管理模式。
- 另一个问题是服从性，能不能正确调用skills工作，而不自作主张想点子。Claude Code经常自作主张，Codex服从性很好。
- 我个人的需求主要包括：1. 不间断地输入论文，进行一个汇总和比较分析，提出一些创新点，这可能需要一个记忆持久化的功能。3. 处理数据，绘制数据图、机制图、流程图等等。4. 然后1和3前面的内容给出一个初稿。此外，还可能做一些遗留项目重构的工作。

- codeg：多CLI管理工具，可一键安装主流CLI（不能识别已安装的CLI，只能识别这个软件安装的），可以读取 Claude Code 默认位置的技能，有预览功能 [xintaofei/codeg: A unified workspace for local AI coding agents (Claude Code, Codex, Gemini CLI, etc.) — desktop app, standalone server, or Docker.](https://github.com/xintaofei/codeg?tab=readme-ov-file)
- AionUi：多CLI管理工具，自带Gemini CLI，更新频率高，预览文件的功能做的不错。问题是只有GeminiCLI支持第三方供应商，反应速度似乎比其他客户端慢，不支持斜杠命令，经常找不到用户导入的技能，只能找到CLI默认位置的技能，需要配合其他工具使用。 [iOfficeAI/AionUi: Free, local, open-source 24/7 Cowork app and OpenClaw for Gemini CLI, Claude Code, Codex, OpenCode, Qwen Code, Goose CLI, Auggie, and more | 🌟 Star if you like it!](https://github.com/iOfficeAI/AionUi)
- Cherry Studio：非常全面的聊天客户端，新增的智能体功能支持自定义角色+多工作区，支持斜杠和@命令，标准.claude文件管理角色技能，一键打开VSCode。问题是导入技能不便，输出偶尔中断，内置的Claude Code比较老旧。
- Hello-Halo：基于Claude Code，反应迅速，支持斜杠命令。问题是MCP尚不可用，且强制全自动执行。 [openkursar/hello-halo: Open-source Claude Code GUI — like Claude Cowork. Visual AI assistant with remote access, file management, and built-in AI browser.](https://github.com/openkursar/hello-halo?tab=readme-ov-file)
- openakita：配置简单，反应快，执行准确。还有活人感引擎（主动询问）、多层次记忆和反思系统（自动总结聊天信息作为长期知识库）。处于开发初期，问题还很多。 [openakita/openakita: An open-source AI assistant framework with skills and agent architecture](https://github.com/openakita/openakita)
- openhanako：界面美观，基于Claude Code，偏向AI伴侣，具有记忆系统。处于开发初期，问题还很多，主要问题是权限不足，甚至无法读取claude技能。 [liliMozi/openhanako: A personal AI agent with memory, personality, and autonomy — built on Electron.](https://github.com/liliMozi/openhanako)
- bitfun：界面简洁美观，支持多CLI，cowork可以完成复杂任务（如调用drawio绘制流程图）。主要问题是不明原因卡死。 [GCWing/BitFun: BitFun is a next-generation AI assistant with built-in Code Agent and Cowork Agent.](https://github.com/GCWing/BitFun)

## Anki自定义安装位置
collapsed:: true
	- ((6909f086-221a-4495-84a3-88c83945dfb7))
- 需求：在Windows系统中安装Anki时只能自定义启动器位置，主程序默认安装在C盘，会占用大量空间，需要移动到其他磁盘
- 根据官方说明，主程序安装目录为`%LOCALAPPDATA%\AnkiProgramFiles`，将其移动到其他位置，然后使用uv（python包管理器）启动即可。但是官方给出的批处理脚本存在问题，我使用以下脚本启动（注意替换主程序目录，使用前先进入主程序目录执行`uv sync`） [Managing Files - Anki Manual](https://docs.ankiweb.net/files.html#running-from-a-flash-drive)
  
  ``` bash
  Set WshShell = CreateObject("WScript.Shell")
  WshShell.Run "cmd /c cd /d d:\Programs\AnkiProgramFiles && uv run anki", 0, False
  ```

- 主要有三种类型：文字->指令，文字->图片，图片->图片
- #+BEGIN_NOTE
  文字转指令
  #+END_NOTE
- gemini-diagram-prompt-zh：skill，lovasz1受到PaperBanana多智能体协作流程启发创作的纯指令skills，输入pdf及参考图，或者输入需要优化的流程图，输出描述流程图内容的绘图指令，然后自己把绘图指令发送给聊天机器人生图。 [lovasz1/cute-PaperBanana-skill: PaperBanana-inspired prompt-only skill for Gemini academic figures](https://github.com/lovasz1/cute-PaperBanana-skill#_prevue)
- academic-figure-generator：skill，输入文档(PDF、DOCX、LaTeX)，生成示意图或架构图绘图指令 [LigphiDonk/academic-figure-generator: AI 驱动的学术论文配图生成平台。上传论文 → AI 分析内容生成 Prompt → 一键生成高质量科研配图，还有配套的skill可在主流agent中使用](https://github.com/LigphiDonk/academic-figure-generator)
- collapsed:: true
  #+BEGIN_NOTE
  文字或图片转图片
  #+END_NOTE
- PaperBanana：输入文字，返回多幅候选图，质量较高，详见“Paper Banana”条目
- AutoFigure-Edit-custom：网页应用，基于AutoFigure-Edit增加了自定义第三方大模型API接口，在配置环境时提示“Failed to build numpy==1.26.4”。在此情况下，启动server.py，由柏拉图AI提供服务，分别测试了gpt-image-1和gemini-3-pro-image-preview，log中均提示未找到返回图片。 [jaxhah2017/AutoFigure-Edit-custom](https://github.com/jaxhah2017/AutoFigure-Edit-custom)
- drawio（drawio-native）：skill，输入文字描述生成drawio格式可编辑的技术路线图，并调用drawio桌面版到处图片，特色是不依赖drawio MCP，多轮优化。适合画简单的流程图。 [Agents365-ai/drawio-skill: Claude Code skill for generating Draw.io diagrams (.drawio XML) and exporting to PNG/SVG/PDF via native draw.io desktop CLI.](https://github.com/Agents365-ai/drawio-skill?tab=readme-ov-file)
- drawio（drawio-next-ai）：skill，输入文字描述或图片，调用next-ai-draw-io mcp server，在drawio网页端生成/复刻/微调可编辑的流程图 [bahayonghang/drawio-skills: drawio agent skill](https://github.com/bahayonghang/drawio-skills/tree/main)
- nsfc-roadmap：skill，输入文字描述，生成drawio格式可编辑的技术路线图，问题是始终无法正确执行其中的python脚本 [nsfc-roadmap - Agent Skill by huangwb8 | SkillsMP](https://skillsmp.com/skills/huangwb8-chineseresearchlatex-skills-nsfc-roadmap-skill-md)

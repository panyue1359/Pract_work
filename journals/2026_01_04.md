- “Phases and Components information”选项卡（以下简称第二步）中元素顺序是**溶质元素在上，溶剂元素排最后**，这个顺序决定了第四步“Material Parameters”-“Phase-Composition Properties”中表格（以下简称第四步表格）Ceq项填写的顺序以及“Composition_析出相名称.csv”文件（以下简称平衡组成文件）中元素的顺序。
  logseq.order-list-type:: number
- 第二步中相的顺序是**析出相在上，基体相在最下**；这决定了“Functional Properties”中“Tdb Phases”和“Phase Map”，以及平衡组成文件中相的顺序。
  logseq.order-list-type:: number
- 根据“Grand_Potential_MPI\\functions”下的“functionF_04.h”文件，平衡组成文件的文件名是Composition_<Phases_tdb[0]>，即第二步中相列表的第一行（第一个析出相的名称），如果写多了会提示“stat: No such file or directory”。
  logseq.order-list-type:: number
	- 平衡组成文件中第一行的名称可以随意写，但顺序必须正确
	  logseq.order-list-type:: number
- 平衡组成文件**第一序是相的顺序，第二序是元素顺序**，即T-元素1@相1-元素2@相1-元素i@相1-元素1@相2-元素2@相2-元素i@相2-元素1@相j-……-元素i@相j
  logseq.order-list-type:: number
- 第四步表格和平衡组成文件中**只填溶质元素，不填溶剂元素**，如果填多了会提示“gsl: interp.c:83: ERROR: x values must be strictly increasing”。
  logseq.order-list-type:: number

- 需求：服务器上查找、移动、编辑文件都不方便，想要在本地电脑使用VScode远程操控
- 系统：CentOS 7.5.1804，Linux内核版本3.10.0-862.el7.x86_64
- 安装Docker（详见“CentOS安装Docker并供普通用户使用”条目），拉取镜像，执行`docker run -d --name=code-server --network=host --privileged --rm -e PUID=自己的uid -e PGID=自己的gid -e TZ=Etc/UTC -e PASSWORD=登陆密码 -e DEFAULT_WORKSPACE=/me lscr.io/linuxserver/code-server:latest`创建容器，在本地电脑浏览器中输入地址打开，速度很快。[linuxserver/code-server - Docker Image | Docker Hub](https://hub.docker.com/r/linuxserver/code-server)
- 如果没有装Docker，可以下载tar.gz包解压后运行“bin/code-server”启动服务端，配置文件为“.config/code-server/config.yaml”。coder的code-server在CentOS上的最高版本是4.16.1。 [coder/code-server: VS Code in the browser](https://github.com/coder/code-server), [云服务器部署VS Code网页版（使用code-server）_哔哩哔哩_bilibili](https://www.bilibili.com/video/BV1w44y1y7S2/#_prevue)
- 安装插件：插件商店不全，需要手动导入.vsix格式的插件 [VSCode插件下载](https://vsc-extension.dreamsoul.cn/) [2025 年 VSCode 插件离线下载攻略：官方渠道一键获取 - 知乎](https://zhuanlan.zhihu.com/p/26003070992#_prevue)

- 需求：在ParaView中添加标签显示OpenFOAM多相流某一相的总体积分数和总网格数的比值（充填率）
- 参考“ParaView显示动态标注”条目，将`CellData['tmp']`修改为`CellData['alpha.Al']`不生效，似乎`inputs[0]`读取不到OpenFOAM的数据
- Ctrl+Space搜索“Python Annotation”，在Expression栏输入以下代码，其中sum函数将Al相的体积分数求和，shape函数获得总网格数，round函数实现保留三位有效数字，头部的格式化字符串实现格式化 [12. Annotations — ParaView Documentation 6.0.0 documentation](https://docs.paraview.org/en/latest/ReferenceManual/annotations.html#python-annotation-filter)
  ``` python
  'Filling Process: %.1f %%' % (round(sum(inputs[0].CellData["alpha.Al"])/shape(inputs[0].CellData["alpha.Al"])[0],3)*100)
  ```

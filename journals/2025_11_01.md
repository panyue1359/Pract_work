- 远端服务器：CentOS 7.5，终端：MobaXterm 25.3，本地：Win 10
- #+BEGIN_PINNED
  x11转发
  #+END_PINNED
- 管理员权限安装依赖：`yum install freeglut freeglut-devel` [Centos7 安装 Paraview5.6.2_centos paraview-CSDN博客](https://blog.csdn.net/hly19990313/article/details/125360220)
  logseq.order-list-type:: number
- 官网下载ParaView 5.13.3的Linux版，上传至服务器，解压
  logseq.order-list-type:: number
- 在MobaXterm交互式终端中进入ParaView的“bin”文件夹，执行`MESA_GL_VERSION_OVERRIDE=3.3 ./paraview`，弹出ParaView窗口（直接执行`./paraview`会报错） [Paraview - 上海交大超算平台用户手册](https://docs.hpc.sjtu.edu.cn/studio/paraview.html#id3)
  logseq.order-list-type:: number
	- 在WindTerm 2.7.0中执行相同命令，会报错，直接执行`./paraview`也会报错
	  logseq.order-list-type:: number
- #+BEGIN_PINNED
  head-less服务端
  #+END_PINNED
- 下载**和本地版本相同**的Linux包上传至服务器，解压后进入“/bin”文件夹，启动“pvserver”，作者使用**5.13.3**。6.0.1版本提示`/lib64/libm.so.6: version GLIBC_2.27 not found`。
  logseq.order-list-type:: number
- 本地电脑打开ParaView，点击“Connect”图标，点击“Add Server”添加服务端，“Name”可自定义，“Host”栏填远端服务器的ip地址，其他保持默认。添加完成点击“Connect”。[ParaView 的远程模式 | Giskard's CFD Learning Tricks](https://xiaopingqiu.github.io/2015/05/03/paraview-remote/)
  logseq.order-list-type:: number
	- 多个客户端连接同一个服务端，只有一个客户端可以控制，其他客户端只能观看 [8. Remote and parallel visualization — ParaView Documentation 6.0.0 documentation](https://docs.paraview.org/en/latest/ReferenceManual/parallelDataVisualization.html#managing-multiple-clients)
	  logseq.order-list-type:: number
	- 使用反向连接，将本地电脑ParaView作为服务端，远程服务器连接提供数据，但是--client-host不会填，执行`arp -a`和`who`查看在线设备ip填入，同样无法连接
	  logseq.order-list-type:: number

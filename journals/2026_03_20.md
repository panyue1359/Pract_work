## Claude Code自定义安装路径
collapsed:: true
	- ((69b41a65-81ef-4c16-ae07-7e37ac741551))
- Claude Code是Anthropic开发的CLI工具，默认安装在C盘。为了节省C盘空间，使用ShadowSmallBaby制作的PowerShell脚本安装在其他位置。[OneToolkit/tools/claude at main · ShadowSmallBaby/OneToolkit](https://github.com/ShadowSmallBaby/OneToolkit/tree/main/tools/claude)
- 直接在PowerShell中运行脚本提示“未对文件 .\\update-cc.ps1 进行数字签名。”：在PowerShell中执行`set-ExecutionPolicy ByPass Process`，然后运行脚本
- 提示有几处语法错误：该脚本需要使用新版PowerShell运行，执行`$psversiontable`查看版本，执行`winget search powershell`查找安装包，执行`winget install Microsoft.PowerShell`查找安装源（需要魔法），执行`winget install Microsoft.PowerShell --source Microsoft`安装（需要魔法），然后使用PowerShell 7运行脚本 [PowerShell 升级和 Windows Terminal 配置 PowerShell - 知乎](https://zhuanlan.zhihu.com/p/401439255)

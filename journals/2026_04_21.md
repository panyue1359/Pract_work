## AI辅助文本校对

collapsed:: true

- 需求：对长文本（上万字）进行标点符号、错别字、语法检查
- GitHub检索，检索词1：`in:readme 错别字 校对 模型`，检索词2：`in:readme 文本校对 模型`
- 工作流程：在Sigil中打开EPUB，复制章节内容到校对工具
- AI Proofreader：VSCode插件，可划选校对，也可拆分长文本逐块校对，使用百炼云qwen3-max模型，效果很好。默认按空行分割，对于使用首行缩进而非每个空行分割的建议手动拆分后使用“按长度拆分”再校对 [Fusyong/ai-proofread-vscode-extension: A VS Code extension for document and book proofreading based on LLM services](https://github.com/Fusyong/ai-proofread-vscode-extension)
- Kouzer：单文件网页应用，配置简单，但效果不稳定 [may3rr/Kouzer: 抠字儿 ，一个极简的 AI 论文、长文本校对工具。单文件静态 HTML 实现，粘贴文本，静候 AI 逐行抠出语病与错别字。](https://github.com/may3rr/Kouzer)
- AI-DocProofreader：python命令行应用，读取docx文件并生成AI批注。实测发现部分代码已失效，经AI修复后可用，但过于吹毛求疵，很多问题提得莫名其妙。 [jack91620/AI-DocProofreader: AI-powered document proofreading tool with Microsoft Word integration for Chinese texts](https://github.com/jack91620/AI-DocProofreader#)
- AutoDocxProofread：桌面应用，但检查力度很弱，即使使用“锱铢必较”模式也检测不出问题 [CZ600/AutoDocxProofread: An intelligent long-document proofreading software powered by large language models. 基于大模型的文档校对软件](https://github.com/CZ600/AutoDocxProofread)

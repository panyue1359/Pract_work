- PaperBanana是一款用于生成科研机制图的工具，基于PaperBananaBench数据集结合五个智能体顺序优化绘图，效果优于直接使用大模型绘图 [dwzhu-pku/PaperBanana: PaperBanana: Automating Academic Illustration For AI Scientists](https://github.com/dwzhu-pku/PaperBanana), [谷歌北大联手学术版Banana爆火，论文图表100%精确生成 - 量子位的文章 - 知乎](https://zhuanlan.zhihu.com/p/2002745910275360349)
- 官方仓库仅支持Gemini API，Mylszd基于官方工具进行了中文化并提供了Evolink接口 [Mylszd/PaperBanana-CN: AI 驱动的学术论文配图生成工具 — 全中文界面，一键安装，国内直连](https://github.com/Mylszd/PaperBanana-CN)，shengnan-Luo基于Mylszd仓库提供了自定义API接口 [shengnan-Luo/PaperBanana](https://github.com/shengnan-Luo/PaperBanana)
- 下载shengnan-Luo的仓库，在Anaconda Prompt中执行以下代码安装运行
  ``` cmd
  conda create -n mydemo python=3.10 -y
  conda activate mydemo
  pip install -r requirements.txt
  streamlit run demo.py --server.port 8501
  ```
- 使用柏拉图AI接口，测试了不同的文本模型和图像模型组合，当前性价比最优的组合是 claude-haiku-4-5-20251001+sora_image。

## Kopia-多端文件备份
collapsed:: true
	- ((65030192-d6e7-4fac-823d-a9ede88df32a))
- 跨平台备份，端到端加密，可直接上传到云端，支持WebDav [github官网](https://github.com/kopia/kopia), [用法详解](https://zhuanlan.zhihu.com/p/351606263)
- 有备份历史，可以恢复到任意时间点 [Features | Kopia](https://kopia.io/docs/features/#backup-files-and-directories-using-snapshots)
- 多端共享 “Multiple users or computers can share the same repository: if different users have the same files, the files are uploaded only once as Kopia deduplicates content across the entire repository.” [Features | Kopia](https://kopia.io/docs/features/#snapshots)
- 可以恢复整个文件夹或者单个文件 [snapshot restore | Kopia](https://kopia.io/docs/reference/command-line/common/snapshot-restore/)
- 右键 Kopia UI 快捷方式-属性，添加`--user-data-dir='full\path'`可修改Kopia配置文件路径 [feature request: make kopia UI portable · Issue #368 · kopia/kopia](https://github.com/kopia/kopia/issues/368#_prevue)

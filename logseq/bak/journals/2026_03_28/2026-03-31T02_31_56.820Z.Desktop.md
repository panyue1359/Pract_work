- ## Win10/11开机跳过锁屏界面
  collapsed:: true
	- ((6909f184-2696-4b56-b3ff-a6d317b005ed))
- 需求：主要的远程控制软件如向日葵、网易UU在解锁锁屏界面后才会启动，无法实现无人值守。同时，又希望保留锁屏功能，允许手动锁屏。
- 在解锁锁屏界面之前启动GUI程序（pre-load）在Windows上是不允许的，因此只能设法跳过这个界面，再启动需要启动的软件 [How to Run GUI Apps as Windows Services | The Core Technologies Blog](https://www.coretechnologies.com/blog/windows-services/gui-applications-as-windows-services/#_prevue)
- 找到注册表中的“HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\PasswordLess\Device”路径，双击右侧`DevicePasswordLessBuildVersion`，2改成0，然后重启。打开`netplwiz`，此时可显示“要使用本计算机，用户必须输入用户和密码”选项，取消勾选该项，即可在开机时自动登录，同时保留锁屏功能。 [Win10/Win11 设置开机自动登录的方法 - richens的文章 - 知乎](https://zhuanlan.zhihu.com/p/624433183)
- ## 锐捷客户端跳过UAC认证提示
  collapsed:: true
	- ((6909f184-2696-4b56-b3ff-a6d317b005ed))
- 需求：锐捷绿色客户端开机启动时会弹出UAC认证窗口，希望跳过这步
- 首先取消勾选锐捷客户端“开机启动”功能，然后尝试：
	- 直接将锐捷客户端根目录下的“RuijieSupplicant.exe”添加到任务计划程序中，使用管理员权限运行，无法正常启动。
	- 使用UAC白名单小工具为“RuijieSupplicant.exe”免UAC快捷方式，然后将快捷方式放到开机启动项文件夹，可以正常启动，不过图标不能正常点亮，是灰色的。 [UACWhitelistTool:📃 UAC 白名单小工具！ - AtomGit | GitCode](https://gitcode.com/gh_mirrors/ua/UACWhitelistTool)
